About

What is Into Frame?

Into Frame is a research project exploring how generative AI can reconstruct spatial depth and geometry from a flat image — allowing users to explore a scene as if they were inside it.

Process

How it works

A brief overview of the pipeline — expand each step as the project develops.

Image input

A single image is uploaded and pre-processed for the generative pipeline.

Depth & geometry estimation

The model infers a dense depth map and reconstructs 3D geometry from the 2D scene.

Environment generation

Generative techniques fill in occluded regions and produce a complete, navigable 3D scene.

Interactive exploration

Users can move freely through the reconstructed environment in real time.
